# Megaline Mobile Plans Analysis / Análisis de planes móviles Megaline

## Project Description / Descripción del proyecto

This project analyzes customer behavior for Megaline, a telecommunications company that offers two prepaid mobile plans: Surf and Ultimate. The goal is to compare user consumption patterns and monthly revenue in order to determine which plan generates more income.

Este proyecto analiza el comportamiento de clientes de Megaline, una empresa de telecomunicaciones que ofrece dos planes móviles de prepago: Surf y Ultimate. El objetivo es comparar patrones de consumo e ingresos mensuales para determinar qué plan genera mayores ingresos.

## Objective / Objetivo

Determine which mobile plan generates higher average revenue and evaluate whether the differences between user groups are statistically significant.

Determinar qué plan móvil genera mayores ingresos promedio y evaluar si las diferencias entre grupos de usuarios son estadísticamente significativas.

- Monthly consumption calculation per user.
- Monthly revenue calculation per customer.
- Exploratory data analysis.
- Revenue comparison between Surf and Ultimate plans.
- Statistical hypothesis testing.

## Main Findings / Principales hallazgos

- Differences were identified in user behavior between the Surf and Ultimate plans.
- Monthly revenue was calculated based on included plan limits and additional charges.
- Statistical tests were applied to compare average revenue between plans and regions.
- The analysis provided data-driven insights to support marketing budget decisions.

---

- El análisis permitió generar información útil para apoyar decisiones de presupuesto publicitario.
